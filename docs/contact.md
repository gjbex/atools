# Contact &amp; support

Bug reports and feature request can be sent to the developer,
[Geert Jan Bex](mailto:geertjan.bex@uhasselt.be), Hasselt University, or
preferably, be submitted as issues on
[GitHub](https://github.com/gjbex/atools).
